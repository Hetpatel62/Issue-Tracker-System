# Issue Tracker System

An intermediate-level full stack application that simulates an internal issue tracking platform.

## Features
- Create issues
- Assign priority
- Update issue status
- RESTful API
- SQLite persistence
- Clean MVC-style backend architecture

## Tech Stack
- Backend: Node.js, Express
- Frontend: HTML, CSS, JavaScript
- Database: SQLite

## How to Run
1. npm install
2. npm start
3. Open frontend/index.html in browser

## Project Structure
- routes → API endpoints
- controllers → business logic
- models → database access
